# Cozanet-docs
Public documentation and development notes for the Cozanet (CZN) project

Cozanet is an early-stage blockchain research and development project exploring
digital settlement and interoperability use cases in emerging markets.

This repository contains public documentation related to the Cozanet project,
including the whitepaper, roadmap, and development updates.

Project Status:
- Smart contract deployed on BNB Smart Chain
- Whitepaper published
- Early-stage research and development
- No live settlement infrastructure beyond the deployed contract

All information is provided for transparency and documentation purposes only.
